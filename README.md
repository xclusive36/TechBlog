# TechBlog

## Description

Full stack MVC application that allows users to create an account, create posts, comment on posts, and edit/delete their own posts and.

It is a Tech Blog where developers can publish their blog posts and comment on other developersâ€™ posts as well. This application follows the MVC paradigm in its architectural structure, using Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication.
